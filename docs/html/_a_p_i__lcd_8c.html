uint8_t  rs 
)
static

Funciones privadas para enviar datos al LCD.

Envía un mensaje al LCD, que puede ser un comando (rs=0) o un dato (rs=1). El envío de mensajes se realiza primero con los 4 bits mas significativos del dato y luego los 4 menos significativos (esto es así porque se trabaja en modo 4BITS). También tiene en cuenta el bit de back_light en cada envío.
