static FSM_STATE_enum handler_busqueda_tarjeta ( )
static

Declaración de funciones privadas para el manejo de cada estado. Cada una devuelve un nuevo estado que es el siguiente estado al que pasa la máquina de estados.

Implementa la lógica para manejar el estado BUSQUEDA_TARJETA. Se encarga de consultarle al lector RFID si detecta una tarjeta, y en caso afirmativo verifica si el UID se encuentra en la base de datos.
